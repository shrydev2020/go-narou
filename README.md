go-narou  downloader and epub generator
===

WHAT
---
Downloader for [小説家になろう](http://syosetu.com/) and generate epub
----

USAGE

1. ./narou init 
2. ./narou https://ncode.syosetu.com/XXX/
3. open the epub using your viewer app
